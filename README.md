# OwlDiary

OwlDiary is a full-stack social journaling platform built as a university capstone project. It gives students a shared digital space to write reflections, publish blog-style posts, customize their profiles, interact through comments and likes, and participate in an admin-moderated campus community.

## Live Demo

Visit the project here:

`https://owldiary.onrender.com`

## What The App Does

- Allows users to create accounts through a registration-code flow
- Supports admin approval for new community members
- Lets users publish rich-text diary and blog-style posts
- Includes likes, comments, and notifications for community interaction
- Supports personalized profiles with themes, typography, accent colors, and background images
- Includes profile gallery uploads and profile customization tools
- Provides admin tools for user approval, registration code management, and platform moderation

## What This Project Demonstrates

- Full-stack application architecture using a React frontend and Express backend
- Authentication and role-based authorization
- REST API design and protected routes
- PostgreSQL data modeling and persistence
- Media uploads and profile/gallery management
- State management for interactive social features like likes, comments, and notifications
- Production deployment across hosted frontend, backend, and database services

## Tech Stack

- React
- Vite
- Express.js
- PostgreSQL
- Node.js
- styled-components
- JWT authentication
- Multer

## Project Focus

This project was designed to demonstrate practical software engineering skills through a real, user-facing platform. The focus was on building a polished full-stack experience with meaningful interactivity, user roles, media handling, and deployable infrastructure rather than just a static prototype.
